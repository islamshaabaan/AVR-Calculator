# AVR-Calculator
  Calculator using AVR-Atmega32, 16x2 LCD, and 4x4 KEYPAD

  Project link: 

  This software allows the user to enter up to 5 digit operands (integers only, no floating point numbers), and can apply 4 different operations.
  # Operations
    1- Addition.
    2- Subtraction.
    3- Multiplication.
    4- Division.
  # KeyPad seems like:
    {'7','8','9','/'}, //Row 0
    {'4','5','6','*'}, //Row 1
    {'1','2','3','-'}, //Row 2   
    {'C','0','=','+'}  //Row 3
